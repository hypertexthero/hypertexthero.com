Eliminate Colour Gremlins  #
Technique to remove colour artifacts from digital images using Photoshop.
  1. Filter → Noise → Median. Set appropriate radius (Around 5 pixels, depending on the image).
  2. Edit → Fade Median. Set Blend Mode to ‘Color’.
