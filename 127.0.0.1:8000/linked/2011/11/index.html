The GitHub Revolution  #

GitHub is truly a system of anarchism, in the most classic sense of the term. It is a system of communication and contribution that is without a central organization or institution of governance. Sure, it is hosted, developed, and maintained by someone but they do not enforce any set of governance or process over the users of the system.

…

The new era, the “GitHub Era”, requires no such internalization of ethos. People and their contributions are as transparent as we can imagine and the direct connection of these people to each other turn social problems back into social problems rather than political problems. The barrier to getting a contribution somewhere meaningful has become entirely social, in other words it is now the responsibility of the community, whether that community is 2 or 2000 people.

A system that enforces its principles without intervention is a tremendous achievement and GitHub’s adoption trend should not be a surprise to anyone.

What Web Browser Should I Use?  #

i) Internet Explorer (IE) is a ‘closed source’ browser, meaning only Microsoft can see how it works, meaning it has many more bugs than other, open source browsers. For some reason (probably Microsoft’s strong-arm marketing and lawyers) it has been a standards browser in the enterprise sector (large companies) for a long time, but has been changing fast.

ii) Firefox, Safari and Chrome are open source browsers which follow web standards. They have fewer bugs and are easier to use. More people in the world use Firefox than IE now.

iii) The web should be accessible to all, regardless of web browser. IE usually breaks things, and web developers despise it for this reason. Best practices are to develop websites for open source browsers which respect web standards FIRST, and THEN code workarounds for IE bugs.

The Social Graph is Neither  #

The funny thing is, no one’s really hiding the secret of how to make awesome online communities. Give people something cool to do and a way to talk to each other, moderate a little bit, and your job is done. Games like Eve Online or WoW have developed entire economies on top of what’s basically a message board. MetaFilter, Reddit, LiveJournal and SA all started with a couple of buttons and a textfield and have produced some fascinating subcultures. And maybe the purest (!) example is 4chan, a Lord of the Flies community that invents all the stuff you end up sharing elsewhere: image macros, copypasta, rage comics, the lolrus. The data model for 4chan is three fields long – image, timestamp, text.

Now tell me one bit of original culture that’s ever come out of Facebook.
