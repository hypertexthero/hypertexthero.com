How to Delete Multiple Photos From an iPad  #

You have transferred many photos from your camera to an iPad using the Apple camera connection kit. There are two ways to delete them:

I. From the iPad itself (hellish):

  1. Open the Photos application.
  2. Touch the ‘All Photos’ tab or ‘Albums’ tab and ‘All Imported’ album.
  3. Touch the window-with-arrow button on the top-right of the screen.
  4. Touch each photo you want to delete until a blue tick mark appears on it. You can also touch and hold your finger down for a second, and then drag across other pictures to select more than one.
  5. Touch the red Delete button near the top-right of the screen.

II. From Mac OS X (better):

  1. Open the Image Capture application.
  2. Select the iPad under ‘Devices’ on the left-hand column.
  3. Select all the photos you would like to delete. You can click on the first photo, scroll down to the bottom, hold down the Shift key and click on the last photo to select them all.
  4. Click the red button near the bottom-left of the screen and click ‘Delete’ in the confirmation box.

Treesaver  #

Javascript framework which uses web standards to dynamically lay out and paginate publications. I don’t like the ‘click to scroll sideways’ user interface that I’m seeing more and more, but anything that promotes web standards instead of proprietary formats is worthwhile, and the Table of Contents idea is nice.

As Zeldman says:

“Mobile is huge. The iPhone, iPad, and Android are huge. On the one hand, they are standards-facing, because they all support HTML5 and CSS3, so you can create great mobile experiences using web standards. You can create apps using web standards.
