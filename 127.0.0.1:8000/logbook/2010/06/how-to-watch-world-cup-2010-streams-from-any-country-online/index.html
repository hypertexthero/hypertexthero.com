How To Watch World Cup 2010 Streams From Any Country Online

Sunday, June 13 2010

Or: How to bypass any geographic-based IP address block and watch any content from anywhere online.

  1. If you’re not using it already, download and install the Firefox web browser.
  2. Go here and click on the green “Add to Firefox” button to install the FoxyProxy Firefox extension. Make sure it is enabled by going to Tools → Addons using the application menu.
  3. Go to SAS Proxy List Pages and find a proxy server located in the country where the content you want to watch is being broadcasted. You can click on the Proxy list sorted by country link. Select and copy the IP address of the proxy.
  4. Add the IP address to the FoxyProxy list:
    1. click on the FoxyProxy icon on the bottom-right of the browser window
    2. click on the ‘Add New Proxy’ button on the window which appears
    3. paste the IP address you copied above into the ‘Host or IP Address’ text input area.
  5. Go to the site where the content you want to watch is being streamed. For example, I’m in Italy watching the world cup matches streamed from Brazilian television at Globo Esporte.

You can now also configure FoxyProxy to automatically use your chosen proxy servers when accessing content from blocked sites, or turn it off when you are done watching by clicking on the FoxyProxy icon on the bottom-right of the window, clicking on the ‘Mode’ drop-down and selecting ‘Completely disable FoxyProxy’.
